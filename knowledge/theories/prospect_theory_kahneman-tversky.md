# Prospect Theory (Kahneman-Tversky)

**Domain:** Economics

**Equation:** `V = Σ π(pᵢ) · v(xᵢ − r), where v(x) = xᵅ if x≥0, −λ(−x)ᵝ if x<0`

**Update Form:** reference_dependent_valuation

**Optimization:** maximize_prospect_value

**Fixed Points:** reference_point_adaptation
